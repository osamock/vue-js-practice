## Install

```bash
$ npm install
$ npm run startAPI
```

## API

```
GET     http://localhost:3000/books
GET     http://localhost:3000/books/1
POST    http://localhost:3000/books
PUT     http://localhost:3000/books/1
DELETE  http://localhost:3000/books/1
```

## DevWebServer

```
$ npm run startDEV
```
